# Scoring de Riesgo Crediticio — PYMEs Industriales

Dashboard interactivo para evaluar el riesgo crediticio de PYMEs industriales.

- Introduce los datos financieros brutos (P&L + Balance)
- Calcula automáticamente todos los ratios: EBITDA, DFN/EBITDA, DSCR, Liquidez, DSO, DPO
- Genera scoring 0–100 con semáforo de riesgo y recomendación de decisión
- Ejemplos precargados: Tubacex 2024, Perfil AA, Perfil C

**Solo para uso interno y análisis orientativo.**
